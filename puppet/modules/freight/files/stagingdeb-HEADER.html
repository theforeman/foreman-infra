The Debian Foreman staging repo

This repo is for building and testing scratch packages before pushing them to the real repos

Every user will have the packages pushed to thier own component, based on the github repository used.
So if you specified 'ares' as the github repo, and 'wheezy' as the distro, your package is at:

deb http://stagingdeb.theforeman.org/ wheezy ares

This repo is also available over rsync at rsync://stagingdeb.theforeman.org/stagingdeb.
